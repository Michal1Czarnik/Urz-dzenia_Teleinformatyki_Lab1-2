# Urządzenia_Teleinformatyki_Lab1-2
W repozytorium znajdują się pliki .pkt, które należy otwierać w programie CISCO Packet Tracer w najnorwszej wersji (8.0). Pliki te zawierają rozwiązania do przykładowych zadań zamieszczonych w stworzonej przez nas instrukcji do zajęć dot. protokołów routingu RIP, OSPF i EIGRP.
